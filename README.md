## 认识 HedgeMemo



你的内容默认保存在这台 Mac 上。只有在截图或自动粘贴需要时，HedgeMemo 才会请你授予相应的系统权限。

## 剪贴板历史

### 复制过的内容，随时找回来

HedgeMemo 会收好复制过的文字、代码、链接和图片，并自动归入相应的分类。输入关键词，就能回到需要的那一条。

- 将内容置顶，或固定到桌面作为随时可见的便签。
- 将剪贴板中的图片直接加入表情包；HedgeMemo 截取的图片会留在独立的“截图”分类。
- 代码会以语法高亮呈现，并可选择喜欢的配色。
- 分类可以开启、关闭和排序；需要更细致的整理时，也可以使用正则表达式建立自己的文本分类。
- 将鼠标停在条目上，即可查看完整内容、来源和收录时间。

桌面固定的内容会从列表第 10 条开始排列；固定多个条目时，先固定的内容会显示在前面。

搜索时加入 `%`，可以按顺序寻找分散在内容中的多个片段。找到目标后，按 <kbd>⏎</kbd> 或点击即可再次复制。

## 表情包管理

### 常用表情，就在鼠标附近

表情包面板既可以从菜单栏打开，也可以通过快捷键出现在鼠标附近。选择一张表情后，面板会自动收起，让你直接回到正在进行的对话。

- 从文件、文件夹或剪贴板加入图片。
- 开启图片捕获后，可以前往其他应用连续收集复制的图片，面板收起也不会中断。
- GIF 会保留动画和原始格式。
- 使用分类、备注和搜索整理收藏，也可以拖动排序或批量移动、删除。
- 通过 ZIP 选择性导入或导出表情与剪贴板内容。

从这里复制的表情不会再次出现在剪贴板历史中。

## 截图与标注

### 捕捉画面，也留下重点

选择一个窗口，或框选屏幕上的任意区域。截图后可以直接使用，也可以先裁剪，并用画笔、箭头、矩形和文字做出标记。

- 调整标注的颜色与粗细。
- 缩放画布，随时撤销或重做。
- 完成的图片会进入剪贴板，同时保存在“截图”分类。
- 可以选择是否在截图后打开编辑器，并记住上一次使用的截图模式。

## 设置与个性化

### 按你的方式使用

快捷键、代码配色、滚动条和启动方式，都可以在设置中调整。

- 分别录制剪贴板、截图和表情包面板的快捷键。
- 决定保存多少条历史，是否保存图片，以及复制后是否自动粘贴。
- 调节菜单栏弹窗以外面板的背景不透明度。0% 保留清晰的玻璃效果，100% 使用完全不透明的系统背景，也可以随时恢复默认。
- 选择是否随登录自动启动 HedgeMemo。
- 在设置中检查更新；有新版本时，菜单栏刺猬会给出轻巧的提示，版本号可直接打开下载页面。

## 快速开始

1. 下载 HedgeMemo，将它放入“应用程序”文件夹并打开。
2. 菜单栏出现刺猬图标后，左键点击可以打开表情包面板；右键点击可以找到剪贴板、截图、导入导出和设置。
3. 按 <kbd>⇧</kbd><kbd>⌘</kbd><kbd>V</kbd> 打开剪贴板历史，选择一条内容即可再次复制。
4. 按 <kbd>⇧</kbd><kbd>⌘</kbd><kbd>E</kbd>，表情包面板会出现在鼠标附近。
5. 打开设置，调整适合自己的快捷键与使用习惯。

## 常用快捷键

| 快捷键 | 用途 |
| --- | --- |
| <kbd>⇧</kbd><kbd>⌘</kbd><kbd>V</kbd> | 打开剪贴板历史 |
| <kbd>⇧</kbd><kbd>⌘</kbd><kbd>P</kbd> | 按当前模式截图 |
| <kbd>⇧</kbd><kbd>⌘</kbd><kbd>E</kbd> | 在鼠标附近打开表情包面板 |
| <kbd>⌘</kbd><kbd>1</kbd> – <kbd>⌘</kbd><kbd>9</kbd> | 取用剪贴板中置顶的内容 |
| <kbd>⏎</kbd> / <kbd>⌫</kbd> | 复制 / 删除当前选中的剪贴板条目 |

剪贴板、截图和表情包面板的快捷键均可在设置中重新录制。

## 权限说明

没有额外权限时，剪贴板历史、表情包管理和手动复制仍然可以正常使用。

| 权限 | 用途 |
| --- | --- |
| 屏幕录制 | 截取窗口或屏幕区域 |
| 辅助功能 | 开启“复制后自动粘贴”后，将已复制内容粘贴到当前应用 |

## 系统要求

macOS 14 或更高版本。
