# MemeMemo

MemeMemo 是一款住在菜单栏里的轻量表情包工具。点击刺猬图标，就能随手收集、整理、搜索和发送常用表情包。

## 现在可以做什么

- 导入图片，或开启捕获后自动收集复制过的图片。
- 用分类整理表情包；可新建、重命名或删除分类。
- 通过备注搜索内容。
- 右键图片可复制、修改备注、移动分类或删除。
- 在管理模式中批量移动、删除图片。
- 按住图片拖动即可排序。
- 导入或导出整套表情包，方便备份和迁移。
- 自动保存复制过的文字和图片，按 `Option + Space` 可以快速找回。
- 在剪贴板历史里点击条目即可重新复制；也可以删除、置顶常用条目。
- 从菜单栏发起截图：可以手动框选，也可以截取鼠标悬浮的窗口。
- 截图完成后会自动保存到表情包，并放入系统剪贴板，方便立即粘贴。
- 截图后可裁剪，并添加画笔、高亮、箭头和文字标注。
- 可在设置里修改剪贴板和截图快捷键；如果两个快捷键相同，会给出提醒。
- 开启自动粘贴时，设置里会提示是否需要允许权限。

## 使用方式

1. 启动 MemeMemo 后，菜单栏会出现刺猬图标。
2. 点击图标打开表情包面板。
3. 点击 `+` 导入图片；点击圆形录制按钮开始或结束自动捕获。
4. 点击图片即可复制，回到聊天软件直接粘贴即可发送。
5. 按 `Option + Space` 打开剪贴板历史，点击文字或图片即可再次复制。
6. 点击相机按钮开始截图；剪贴板和截图快捷键都可在设置里修改。

## 隐私

内容默认只保存在你的电脑中。需要使用截图或自动粘贴时，MemeMemo 会先说明所需权限。
