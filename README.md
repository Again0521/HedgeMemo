## 把常用内容留在手边

HedgeMemo 是一款仅2MB大小的小工具。它可以是你记录复制过的文字、图片的剪切板、可以是表情管理器、也可以是便捷的截屏工具，有了它你不必为了找回一段文字、翻一张表情包，或截下一块屏幕而在多个应用之间来回切换：需要时，点一下菜单栏里的刺猬或按下快捷键就好。

内容默认只保存在你的 Mac 上；需要截图、自动粘贴等权限时，HedgeMemo 会先征求你的同意。

## 剪切板

复制过的内容不再只有一次机会。

- 自动收好文字、代码、链接和图片，并按类型分类。
- 用 <kbd>⇧</kbd><kbd>⌘</kbd><kbd>V</kbd> 随时打开历史记录；**搜索几下就能找到很久以前复制的内容**。
- 鼠标停在条目上即可查看完整内容、来源、收录时间和使用记录。
- 可以将复制过的文本单独固定到桌面，**像便签一样随时查看**。
- 剪切板图片可右键加入表情包库；自己截的图会单独放进“截图”分类，方便回找。
- 支持自动识别并分类代码、链接、普通文本，**支持代码语法高亮**，就像在编辑器一样。
- 支持正则表达式分类复制过的文本。
- 每个分类都能单独开启、关闭和排序。关闭分类时，分类里的记录会一起清理，列表也会更清爽。

## 表情包管理

聊天时最常用的图片，应该一点就能拿到。

- 点击捕获后，可以捕获接下来复制的所有图片，直到你结束捕获。
- 支持全图片格式，GIF 会按动图保存，不会被压成静态图。
- 普通界面里按住缩略图直接拖动即可排序；拖到两张图之间时就会落在那个位置。
- 可按分类整理、搜索备注，并支持多选后批量移动或删除。
- 点击图片会复制到剪切板，切回聊天窗口直接粘贴。
- 导入和导出都使用 ZIP：导出时可选择具体分类；导入时会先读出 ZIP 里的分类，再由你勾选要带回来的内容。

## 截屏

看见想留下的内容，顺手截下来、标一下、马上发出去。

- 默认使用“智能窗口”模式，自动识别窗口并截屏，也可以框选任意区域；快捷键默认为 <kbd>⇧</kbd><kbd>⌘</kbd><kbd>P</kbd>。
- 截图后可裁剪、画笔涂鸦、加箭头、矩形和文字；颜色与粗细都能调整。
- 编辑窗口支持缩放、撤销和重做，完成后图片会立刻进入剪切板，可直接粘贴。
- HedgeMemo 自己截下的图片只保存在“截图”分类，不会和普通复制的图片混在一起。

## 快速开始

1. 打开 HedgeMemo，菜单栏会出现刺猬图标。
2. 点图标打开表情包面板；右键图标可打开剪切板、截图、设置和导入导出。
3. 按 <kbd>⇧</kbd><kbd>⌘</kbd><kbd>V</kbd> 找回最近复制过的内容。
4. 在设置里按自己的习惯调整快捷键、剪切板保存数量、分类和代码配色。

## 常用快捷键

| 快捷键 | 用途 |
| --- | --- |
| <kbd>⇧</kbd><kbd>⌘</kbd><kbd>V</kbd> | 打开剪切板历史 |
| <kbd>⇧</kbd><kbd>⌘</kbd><kbd>P</kbd> | 开始截图 |
| <kbd>⌘</kbd><kbd>1</kbd> – <kbd>⌘</kbd><kbd>9</kbd> | 取用剪切板内置顶的条目 |
| <kbd>⏎</kbd> / <kbd>⌫</kbd> | 复制 / 删除当前选中的剪切板条目 |

快捷键都可以在设置中改成你更顺手的组合。
