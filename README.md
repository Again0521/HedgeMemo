## 一句话认识 HedgeMemo

HedgeMemo 是一款 macOS 菜单栏小工具。它会帮你收好复制过的文字和图片，管理常用表情包，并提供随手可用的截图与标注功能。需要时点一下菜单栏里的刺猬，或按下快捷键，就能把内容找回来、复制出去或继续编辑。

所有内容默认保存在你的 Mac 上。涉及屏幕录制或自动粘贴时，HedgeMemo 会说明所需权限，由你决定是否开启。

## 快速开始

1. 打开 HedgeMemo，菜单栏会出现刺猬图标。
2. 左键点击图标打开表情包面板；右键点击可进入剪贴板、截图、设置以及导入导出功能。
3. 按 <kbd>⇧</kbd><kbd>⌘</kbd><kbd>V</kbd> 打开剪贴板历史，选择一条内容即可再次复制。
4. 在“设置”中按自己的习惯调整快捷键、保存数量、分类、代码配色和启动方式。

## 剪贴板历史

复制过的内容不再只有一次机会。

- 自动记录文字、代码、链接和图片，并按内容类型整理。
- 使用搜索快速找回旧内容；文字、代码和链接都可直接再次复制。
- 鼠标停留在条目上可查看完整内容、来源、收录时间和使用情况。
- 常用内容可以置顶，也可以固定到桌面，像便签一样随时查看。
- 图片可直接加入表情包库；通过 HedgeMemo 截取的图片会单独归入“截图”分类。
- 支持代码语法高亮，让复制的代码更容易阅读。
- 分类可单独开启、关闭、排序；也可以用正则表达式创建自己的文本分类。

### 找回一段内容

按 <kbd>⇧</kbd><kbd>⌘</kbd><kbd>V</kbd> 打开历史记录，输入关键词搜索，选择目标条目后按 <kbd>⏎</kbd> 或点击即可复制。若开启了“复制后自动粘贴”，系统会在获得辅助功能权限后将内容粘贴到当前应用。

## 表情包管理

把聊天里常用的图片放在顺手的位置。

- 从文件、文件夹或剪贴板添加图片，也可以开启捕获，连续收集随后复制的图片。
- GIF 会以动图形式保留；点击表情即可复制原图到系统剪贴板。
- 支持创建分类、添加备注、搜索、拖动排序，以及多选后批量移动或删除。
- 支持 ZIP 导入和导出，导出时可选择分类；导入时可勾选需要带回的内容。

### 添加一张表情

打开表情包面板后，选择导入图片；也可以先开启图片捕获，再在其他应用中复制图片。之后点一下表情，切回聊天窗口粘贴即可发送。

## 截图与标注

看到值得留下的内容，顺手截下来、标一下、立即使用。

- 默认提供智能窗口截图，也可手动框选任意区域。
- 可裁剪图片，使用画笔、箭头、矩形和文字进行标注，并调整颜色和粗细。
- 支持缩放、撤销和重做；完成后图片会进入剪贴板，可直接粘贴到其他应用。
- 可在设置中选择截图后的行为，例如是否自动打开编辑器、是否记住上一次使用的模式。

### 截一次图

按 <kbd>⇧</kbd><kbd>⌘</kbd><kbd>P</kbd>，按当前设置开始截图。完成选择后可立即编辑；若不需要标注，截图会直接进入剪贴板。

## 设置与个性化

在右键菜单的“设置”中，你可以：

- 调整剪贴板历史保存数量、图片保存和分类规则。
- 录制自己的剪贴板、截图与表情包面板快捷键。
- 选择代码高亮配色，显示或隐藏滚动条。
- 开启登录时自动启动。
- 开启“复制后自动粘贴”。首次使用时需要在 macOS 中允许辅助功能权限。
- 查看当前版本、手动检查更新；发现新版本时可直接点击版本号前往下载页面。

## 常用快捷键

| 快捷键 | 用途 |
| --- | --- |
| <kbd>⇧</kbd><kbd>⌘</kbd><kbd>V</kbd> | 打开剪贴板历史 |
| <kbd>⇧</kbd><kbd>⌘</kbd><kbd>P</kbd> | 按当前模式截图 |
| <kbd>⇧</kbd><kbd>⌘</kbd><kbd>E</kbd> | 在鼠标附近打开表情包面板 |
| <kbd>⌘</kbd><kbd>1</kbd> – <kbd>⌘</kbd><kbd>9</kbd> | 取用剪贴板中置顶的内容 |
| <kbd>⏎</kbd> / <kbd>⌫</kbd> | 复制 / 删除当前选中的剪贴板条目 |

快捷键均可在设置中重新录制。

## 权限说明

HedgeMemo 只在需要时请求权限：

| 权限 | 用途 |
| --- | --- |
| 屏幕录制 | 截取窗口或屏幕区域 |
| 辅助功能 | 在开启“复制后自动粘贴”后，将已复制内容粘贴到当前应用 |

不授予权限也不影响剪贴板历史、表情包管理和手动复制等基本功能。
