# HedgeMemo

**住在菜单栏里的小刺猬，帮你随手收好每天要用的图、文字和截图。**

复制过的、截下来的、常用的——都在一个刺猬图标里，需要时一步取回。

</div>

---

## 这是什么

HedgeMemo 是一款常驻在 Mac 菜单栏的小工具。它把三件每天都在做、却总是很麻烦的事，收进了一个安静的刺猬图标里：

| 🦔 表情包库 | 📋 剪贴板历史 | ✂️ 截图 |
| --- | --- | --- |
| 收藏、整理、搜索常用表情包，聊天时一点即发。 | 自动记住你复制过的文字、图片和链接，随时找回。 | 框选或点窗口即可截图，还能随手标注。 |

三件事各自独立，也能配合：截的图能存进剪贴板，剪贴板里的图能一键收进表情包库。

---

## 🦔 表情包库

再也不用在相册里翻半天找那张“狗头”。

- **随手收集**：拖进图片，或开启自动收集，复制过的图片会自己进库；GIF 保持动图不变。
- **分门别类**：新建分类，把表情包按心情、场景归类，想改名、删除都可以。
- **一搜就到**：给图片写个备注，之后用关键词就能找到它。
- **想怎么排就怎么排**：按住拖动排序，落点清清楚楚，还能一键拖到最后。
- **一点即发**：点一下图片就复制好了，回到聊天窗口直接粘贴。
- **批量整理**：进入管理模式，一次移动或删除多张。
- **搬家不丢**：把整个库连同分类打包带走，换台电脑也能原样导入。

## 📋 剪贴板历史

复制了新内容，旧的就找不回来了？这里全都留着。

- **自动记录**：你复制过的文字、图片、链接，它都默默存好。
- **分类清晰**：文本、代码、链接、图片自动归类，还能用你自己的规则筛出想要的那类。
- **随时找回**：按 <kbd>⇧</kbd><kbd>⌘</kbd><kbd>V</kbd> 打开，点一下就重新复制。
- **看清再用**：把鼠标停在条目上，就能看到完整内容、来源和用过几次。
- **常用置顶**：把高频内容钉在最前，<kbd>⌘</kbd><kbd>1</kbd>–<kbd>9</kbd> 直接取用。
- **一键收藏**：看到喜欢的图，右键就能存进表情包库。

## ✂️ 截图

看到什么想留住，抬手就截。

- **两种方式**：自己框选任意区域，或把鼠标移到某个窗口上直接截它。
- **随手标注**：截完可以裁剪，加画笔、高亮、箭头、矩形和文字，颜色和粗细都能挑。
- **马上能用**：截完直接进剪贴板，切回去就能粘贴；也会归到剪贴板的“截图”分类里备查。

## 快速上手

1. 打开 HedgeMemo，菜单栏会出现一只小刺猬。
2. **点图标**，打开表情包面板——拖入图片，或开启自动收集。
3. **点图片**就复制好了，回聊天窗口粘贴即可发送。
4. 按 <kbd>⇧</kbd><kbd>⌘</kbd><kbd>V</kbd> 打开剪贴板历史，找回复制过的任何内容。
5. **右键菜单栏图标**，即可截图、导入导出、打开设置。

## 常用快捷键

| 快捷键 | 作用 |
| --- | --- |
| <kbd>⇧</kbd><kbd>⌘</kbd><kbd>V</kbd> | 打开剪贴板历史 |
| <kbd>⌘</kbd><kbd>1</kbd> – <kbd>⌘</kbd><kbd>9</kbd> | 取用置顶的剪贴板条目 |
| <kbd>⏎</kbd> / <kbd>⌫</kbd> | 复制 / 删除选中条目 |
| <kbd>⌘</kbd><kbd>P</kbd> | 置顶 / 取消置顶 |

> 剪贴板和截图的快捷键都能在设置里改成你顺手的组合。

## 想调点什么

打开设置，你可以：

- 改剪贴板和截图的快捷键。
- 设置剪贴板最多保存多少条、要不要保存图片。
- 开关任意剪贴板分类，或用自己的规则新建分类。
- 开启“复制后自动粘贴”，省下一次粘贴动作。

## 你的东西，只在你的电脑里

HedgeMemo 收集的内容默认只保存在本机，不会上传到任何地方。用到截图或自动粘贴时，它会先告诉你需要哪项系统权限，由你决定是否允许。
